Apresentação

Desenvolvimento e Implantação de Sistema de Gestão da Produção fev de 2018 – jul de 2018

Sistema desenvolvido para controle da produtividade de equipamentos.
Este projeto foi desenvolvido full stack, com embasamento nos indicadores de produtividade do TPM, envolvendo as seguinte etapas:

  • Análise de requisitos
  • Pesquisa de conceitos aplicáveis ao problema
  • Adaptação dos conceitos ao requisitos e regras de negócio
  • Desenvolvimento do Banco de Dados
  • Desenvolvimento full stack
  • Testes, validações e correções
  • Treinamento de usuários
  • Apresentação do Sistema para Diretores, Gerentes e Supervisores
  • Implantação do Sistema
  • Orientações e treinamento de gestores para tomadas de decisão

Os conceitos e indicadores desenvolvidos foram:

  • Total Productive Maintenance
  • OEE (Overall Equipment Effectiveness
  • Indicador de Utilização de Equipamento
  • Indicador de Disponibilidade de Equipamento
  • Indicador de Qualidade
  • MTBF (Mean Time Between Failures)
  • MTTR (Mean Time to Repair)

Tecnologias empregadas

  • Banco de dados MongoDB
  • JQuery, Moment.js e Chart.js
  • PHP, HTML e CSS
  • Ajax
  • Bootstrap
